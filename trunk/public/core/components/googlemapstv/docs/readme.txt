--------------------
3PC: Google Maps TV
--------------------
Version: 1.0
Since: April 8th, 2010
Author: Shaun McCormick <shaun@collabpad.com>

A custom Output Render type for TVs, that renders a Google Map from a TV value.

Usage:
------
Just create a TV and specify "googlemaps" as a TV output. You can use whatever type you want as the input.

Here are the possible values for your TV.

Display an address:
123 Merry Way

Display directions in the map:
123 Merry Way, Blissville, AR to 456 Regrets Ave, Detroit, MI

When streetview is on, adjust the yaw/pitch of an address:
123 Merry Way::5::23
